

# 🎨 Background Color Changer

A simple and interactive React app that lets users change the background color of the page using a floating control panel.

---

## 🚀 Features

* 🎯 Change background color with a single click
* 🎨 Multiple color options (Red, Green, Blue, Pink, Purple, Lavender)
* 💫 Smooth transition effect
* 📱 Responsive floating panel UI
* ⚡ Built with modern React + Tailwind CSS

---

## 🛠️ Tech Stack

* ⚛️ React
* 🎨 Tailwind CSS
* ⚡ Vite

## 🧠 How It Works

* Uses React `useState` to manage the background color
* Button clicks trigger state updates
* The main container dynamically updates its background using inline styles
* A fixed-position panel keeps controls accessible at the bottom

---

## 📂 Project Setup

```bash
# Clone the repo
git clone https://github.com/aadityajoshi2k5/bg-color-changer.git

# Navigate into the project
cd bg-color-changer

# Install dependencies
npm install

# Run the development server
npm run dev
```
